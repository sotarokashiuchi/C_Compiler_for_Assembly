# ABI Function Calling Sequence
- rbp, rbx r12 ~ r15のレジスタは関数終了後に呼び出し前の値と同じでなければいけない
- FLAFSレジスタのDFは関数開始時と終了時に順方向に設定する必要がある

スタックフレーム
- 入力引数領域の終わりは16bitアライアンスに揃えられる
- レッドゾーン

引数
- INTEGERクラス RAX:RDX funx(RDI, RSI, RDX, RCX, R8, R9)
- SSEクラス xmm0:xmm1(xmm0, xmm1, xmm2, xmm3, xmm4, xmm5, xmm6, xmm7)
- 可変長引数の場合,raxレジスタに引数の数を入れる必要がある?
